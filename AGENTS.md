# AGENTS.md

## Cursor Cloud specific instructions

### Overview

First Principles is a Next.js 16 (App Router) web application for spine surgery appropriateness review. Backend is Supabase (cloud-hosted PostgreSQL + Auth + Storage + Realtime). See `README.md` for full project details.

### Environment Variables

A `.env.local` file is required at the project root with:
- `NEXT_PUBLIC_SUPABASE_URL` — Supabase project URL
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` — Supabase anonymous/public key
- `SUPABASE_SERVICE_ROLE_KEY` — Service role key (for admin scripts only)

These are injected from Cursor Cloud secrets. The update script auto-generates `.env.local` from environment variables on startup.

### Standard Commands

All standard dev commands are in `package.json`:
- `npm run dev` — Start Next.js dev server on port 3000
- `npm run build` — Production build
- `npm run test` — Unit tests (Vitest)
- `npm run lint` — ESLint (source in `src/`; lint also picks up `playwright-report/` if present — those warnings are not from source code)

### Gotchas

- **No local Supabase**: There is no `supabase/config.toml`, so `npx supabase start` is not configured. All database operations go to the cloud Supabase instance.
- **Lint noise from `playwright-report/`**: If a `playwright-report/` directory exists (generated by Playwright), ESLint will lint its bundled JS and report many warnings. Run `npx eslint src/` to lint only source code. The pre-existing source lint warnings/errors are in the repo and are not regressions.
- **Demo users**: The app ships with demo user scripts in `scripts/`. Default password for demo users is `Demo2024!`. See `scripts/README.md` for details.
- **Playwright tests**: `@playwright/test` is a devDependency. Browsers are not installed by default; run `npx playwright install` before running E2E tests.
